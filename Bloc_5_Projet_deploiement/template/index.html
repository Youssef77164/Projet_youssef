Description de l'API Youssef sur le Vin

Comment utiliser l'API et a quoi sert t-elle ?

Cette API est un test de déploiement d'un modèle de machine learning dont l'objet est l'estimation de la qualité d'un produit en l'occurrence le vin ici et ce en fonction de différents facteurs.

Pour cela il faut entrer les valeurs correspondantes aux variables explicatives suivantes :

  • fixed acidity
  • volatile acidity
  • citric acid
  • residual sugar
  • chlorides
  • free sulfur dioxide
  • total sulfur dioxide
  • density
  • pH
  • sulphates
  • alcohol

Une fois toutes ces variables numériques entrées, l'API lancera une prediction sur les données choisies et vous obtiendrez l'estimation de la qualité du Vin d'une note entre 3 à 9 sur la base d'échantillons physico-chimiques.


Mode d'emploi:

1. Remplacer les variables explicatives ci-dessous par des valeurs numériques

Mon_input = ['fixed acidity', 'volatile acidity', 'citric acid','residual sugar', 'chlorides', 'free sulfur dioxide','total sulfur dioxide', 'density', 'pH', 'sulphates', 'alcohol']

exemple : Mon_input = [7.0, 0.27, 0.36, 20.7, 0.045, 45.0, 170.0, 1.001, 3.0, 0.45, 8.8]

2. Lancez la requête API ci-dessous:

response = requests.post("https://my-app-youssef-vins.herokuapp.com/quality", json={"quality": Mon_input})

3. Affichez le résultat (par exemple):

print(response.json())

4.résultat final:

{'Niveau de qualité estimé' : 6.0}
